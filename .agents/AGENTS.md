# Design Rules

## UI Design System & Aesthetic Guidelines
Whenever designing web pages, components, or UI elements, always adhere strictly to the following design format:

### 1. Color Palette
- **Primary**: `#007AFF` (Vibrant Blue)
  - Usage: Primary buttons, active navigation states, primary focus states, key UI highlights.
- **Secondary**: `#5856D6` (Royal Violet / Purple)
  - Usage: Secondary actions, badges, accents, and category tags.
- **Tertiary**: `#D75800` (Vibrant Amber / Orange)
  - Usage: Warning states, highlights, quick-action indicators, and creative accents.
- **Neutral / Background**: `#F2F2F7` (Light Off-White / Soft Gray)
  - Usage: Background grids, search bars, inputs, card containers, and secondary text colors.

### 2. Typography
- **Font Family**: `Inter` (sans-serif)
- **Key Hierarchy**:
  - **Headline**: Inter, bold/semibold weight, clean spacing.
  - **Body**: Inter, normal weight, highly readable.
  - **Label**: Inter, medium/semibold weight, smaller sizing.

### 3. Layout & Containers (Cards)
- **Border Radius**: Use rounded corners for containers (`12px` to `24px` for cards/panels).
- **Backgrounds**: Soft, flat backgrounds using Neutral `#F2F2F7` or clean white `#FFFFFF`.
- **Aesthetics**: Clean, modern flat design with generous padding, clean grid alignment, and rounded structures.

### 4. Components & Interactive Elements
- **Buttons**:
  - **Primary**: Filled with `#007AFF`, white text.
  - **Secondary**: Light neutral/gray background, dark text.
  - **Inverted**: Dark gray/black background, white text.
  - **Outlined**: Transparent/white background, thin neutral border, dark text.
  - **Border Radius**: Buttons should have a border-radius of `8px` to `12px`.
- **Search Inputs**:
  - Soft neutral background, borderless, rounded corners (around `8px`), search icon left-aligned.
- **Navigation Capsule**:
  - Rounded capsule navigation bar (segmented control style) with light neutral background, using circular filled icons for the active state (utilizing Primary `#007AFF`).
- **Action Buttons / Chips**:
  - Circular buttons or pill-shaped chips with clear functional icons (e.g. pencil for edit, trash for delete).
